# Termux Bootstrap Example

This directory contains a lightweight Termux bootstrap example derived from the local phone-debugging workspace.

Use it as a starting point when you want a fresh Termux shell to look more like a small Codex workstation.

Key file:

- `termux-dev-bootstrap.sh`

The bootstrap sets up:

- a stronger shell profile
- a few common aliases
- Java / Android SDK environment defaults
- a simple `devcheck` helper

This is intentionally an example, not a one-size-fits-all production installer.
